**Квадратное уравнение** — уравнение вида 
$$ax^2+bx+b^2=0$$
при 
$$a\neq 0$$
**Неполное квадратное уравнение**:
1) $$ax^2=0$$

имеет один корень
$$x = 0$$
2) $$ax^2 + c = 0$$
имеет два корня — противоположные цифры, или не имеет корней

3 $$ax^2 + bx = 0$$
всегда имеет два корня, один из которых равен 0



**Теорема Виета**: если у квадратного уравнения 
$$ax^2 + bx + c = 0$$
существуют корни, то:
$$\begin{cases}x_1 + x_2 = -\frac{b}{a},\\ x_1 * x_2 = \frac{c}{a} \\\end{cases}$$
Если a = 1, тогда:
$$\begin{cases}x_1 + x_2 = -b,\\ x_1 * x_2 = c \\\end{cases}$$

Если a = -1, нужно домножить на -1.
