**Операторы сравнения:** равно **=**, не равно **<>**, больше **>**, меньше **<**, больше или равно **>=**, меньше или равно **<=**.

```sql
SELECT title, price 
FROM book
WHERE price < 600;
```

```sql
SELECT title, author, price * amount AS total
FROM book
WHERE price * amount > 4000;
```

```sql
SELECT author, title, price
FROM book
WHERE amount < 10;
```
