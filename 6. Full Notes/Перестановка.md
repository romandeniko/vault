Способ последовательного расположения объектов с учетом порядка.

Для n объектов существует n! способов расположения. Это число называют P.

Число объектов называют длиной перстановки.

Перестановку, где все объекты стоят на своих местах, называют тождественной.

Перестановку, у котрой каждый элемент находится не на своем месте, называют беспорядком.

Число всех беспорядков длины n обозначают через субфакториал !n. 

#### Формулы общего случая

$$P_n = n * P_(n-1)$$
$$P_(n+1) = (n+1)P_n$$
$$P_n=P_(n-1)*n$$
$$P_n=(n-1)*(P_(n-1)+P_(n-2))$$


#### Формула для определения беспорядка

$$!n=(1-n)*(!(n-1)+!(n-2))$$
