**Счётным** называется бесконечное множество, которое равномощно N, то есть элементы которого можно пронумеровать.

Само множество N  равномощно себе.

Множество Z также является счетным.

### Теоремы

```
Любое бесконечное подмножество счётного множества является счётным множеством.
```

```
Множество рациональных чисел счётно.
```
